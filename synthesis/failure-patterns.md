# Failure Patterns

This artifact tracks recurring failure, decline, collapse, bankruptcy, strategic-error, and leadership-error patterns.

## Use

Add to this file when a failure pattern appears across multiple targets or when a single failure case identifies a pattern worth testing against future targets.

## Failure Pattern Entry Template

```markdown
## Failure Pattern: <Name>

### Summary

### Targets observed

### Warning signs

### Causes

### Decisions that worsened the failure

### Capital or financial signals

### Leadership or culture signals

### What might have prevented or slowed the failure

### Lessons to avoid

### Open questions
```
