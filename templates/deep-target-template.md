# Deep Target Template

Use this template when a target needs extensive, comprehensive, professional reports on each pillar and subtopic.

A deep target should be organized as a navigable local file tree, not as a single large markdown file.

---

## Required top-level files

```text
INDEX.md
profile.md
timeline.md
historical-context.md
companies.md
business-and-organization.md
leadership-and-human-systems.md
capital-and-wealth.md
decision-making-and-judgment.md
lessons-to-copy.md
lessons-to-avoid.md
sources.md
```

The top-level pillar files may act as summaries or landing pages. Detailed research should live in pillar subdirectories.

---

## Required pillar directories

```text
business-and-organization/
leadership-and-human-systems/
capital-and-wealth/
decision-making-and-judgment/
lessons/
sources/
```

Each directory must contain its own `INDEX.md` listing only the files directly inside that directory.

---

---

## Depth rule

A subtopic may be expanded into additional files or nested directories when one file becomes too large or mixes multiple distinct subjects.

When this happens:

1. Create a new subdirectory with a clear name.
2. Add a local `INDEX.md` to that subdirectory.
3. Move or create the relevant subtopic reports there.
4. Update only the affected local indexes.

---

## Research standard

Deep reports must remain source-grounded and must separate:

- fact;
- participant narrative;
- reporting;
- interpretation;
- model-based financial judgment;
- disputed claims;
- uncertainty;
- and open questions.
