## Intended contents

Each company target should receive its own directory named with a clear, stable, lowercase, hyphenated name.

Example:

```text
mcdonalds/
nvidia/
starbucks/
walmart/
```

Each company directory should follow the target template unless a specific case requires a justified variation.

## Rule

This index lists only files and directories directly visible inside `targets/companies/`.
